Virtual Memory(가상 메모리)

실제 메모리의 용량에서 가상으로 용량을 확장하는 기술이다. 확장된 용량은 보조기억장치의 일부분이다.

MMU(Memory Management Unit)이 가상 주소를 물리적인 주소로 변환 해준다.

Page fault(페이지 폴트)

접근하는 데이터가 가상 메모리 영역에만 있고 실제 메모리 영역에 없는 것을 Page fault라고 한다.

Demand Paging(요구 페이징)은 CPU가 요청할 때 프로세스의 데이터를 메모리에 올리는 것을 말한다. 이로써 Page fault가 해결된다.

페이지 교체 알고리즘

페이지는 한정된 메모리 영역으로 인해 교체가 이루어져야 한다. 이때 몇가지 방법이 있다.

1) FIFO(First In First Out)
2) LRU(Least Recently Used) - 최근 사용되지 않은 페이지를 교체한다.

페이지 교체 범위

페이지 교체 타겟의 범위

1) Global(모든 프로세스)
2) Local(자기 프로세스)
*Global이 더 효율적이라고 한다.
