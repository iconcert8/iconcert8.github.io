Thread

프로세스 내에서 실행되는 제어흐름 단위이다.
*Daemon thread - 메인 스레드에 종속적인 스레드이다. 메인 스레드가 종료되면 강제 종료된다.

Thread 클래스

Java에서 스레드를 생성하기 위해 사용하는 클래스이다.

사용 방법은 다음과 같다. 
public static void main(String[] args) {
    Thread thread = new Thread(() -> {
        System.out.println("Hello world");
    });
    thread.start();
}

Runnable 인터페이스

스레드에서 실행하는 함수의 인터페이스이다. Thread 클래스도 Runnable을 상속받은 클래스이다.
상속을 받아 사용해도 되나, 무명 클래스로 오버라이드 해서 사용을 많이 한다.

위 코드예제에도 이미 나와있다. 아래 람다코드를 인자로 바로 넣게되면 위 Thread 예제와 같다. 
public static void main(String[] args) {
    Runnable runnable = () -> System.out.println("Hello world");
    Thread thread = new Thread(runnable);
    thread.start();
}
