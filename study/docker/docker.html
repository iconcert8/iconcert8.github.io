Docker

컨테이너 기반의 소프트웨어 개발 플랫폼이다. 컨테이너를 제공/실행 시켜준다. Docker는 Server-Client 구조로 CLI 명령 또한 서버 API를 호출하는 방식이다.
🛤️ Virtualization(가상화) 컨테이너 설명 참조

Docker 특징

오버레이 네트워크

물리 네트워크 위에 성립되는 가상의 네트워크이다.

유니온 파일시스템

여러개의 파일 시스템을 하나의 파일 시스템으로 마운트 하는 기능이다.

중복되는 파일 레이어에 대해서 다운로드를 건너뛴다.

상위 레이어에서 하위 레이어에 쓰기 작업을 할 경우 하위 레이어를 복사하여 쓰기작업을 한다(COW, Copy On Write).

Docker는 위의 2가지 특징 기능을 활용해 컨테이너의 네트워크와 파일 시스템을 구성하여 효율적으로 환경을 구성할 수 있게 만들어 준다.

그리고 Dockerfile을 활용한 커스텀 이미지(컨테이너 틀) 제작을 가능케 하고, docker-compose로 다중 컨테이너 설정 작업을 지원하여 개발 편의성을 높인다.
