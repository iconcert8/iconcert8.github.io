OAuth

다른 서비스의 인증정보로 접근권한을 부여 받을 수 있는 표준 프로토콜이다.
구글 로그인, 페이스북 로그인 기능이 바로 OAuth이다.

동작 방식

1) 유저가 클라이언트 서비스에 로그인 요청

2) 클라이언트 서비스에서 인증 서버(구글, 페이스북)에 로그인 요청

3) 인증 서버에서 유저에게 로그인 페이지 제공

4) 유저는 인증 서버의 로그인 페이지로 로그인

5) 인증 서버는 유저에게 Authorization Code 제공

6) 클라이언트 서비스로 리다이렉트, Authorization Code 클라이언트 서비스에 전달

7) 클라이언트 서비스는 Authorization Code, Client ID, Client Secret을 가지고 인증 서버에게 Accsess Token 요청

8) 클라이언트 서비스는 응답 받은 Access Token을 DB에 저장

9) 유저 로그인 성공

10) 유저가 클라이언트 서비스에 리소스 요청

11) 클라이언트 서비스는 유저와 매핑된 Access Token을 가지고 리소스 서버에 요청

12) 리소스 응답
